### [Blog 1 - Announcing the Regional 2025 AWS Partners of the Year for Europe, Middle East and Africa](3.1-Blog1/)

This blog announces the 2025 AWS Partners of the Year for the Europe, Middle East and Africa (EMEA) region. These annual awards recognize AWS Partner Network (APN) members who consistently deliver outstanding results and drive innovation for our customers.

### [Blog 2 - How to customize responses to layer 7 DDoS attacks using AWS WAF Anti-DDoS AMR](3.2-Blog2/)

This blog explains how AWS WAF Anti-DDoS AWS Managed Rules (Anti-DDoS AMR) works and how you can customize its behavior using labels and additional AWS WAF rules. You'll be guided through three real-world scenarios, each illustrating a different customization technique.

### [Blog 3 - Build AI agents for Amazon Bedrock using the AWS CDK and deploy with GitHub Actions](3.3-Blog3/)

This blog shows how to build AI agents for Amazon Bedrock using the AWS Cloud Development Kit (CDK) and deploy them using GitHub Actions. It provides a complete end-to-end solution for creating, testing, and deploying AI agents in a production environment.

### [Blog 4 - S&P Global data integration expands Amazon QuickSight Research capabilities](3.4-Blog4/)

This blog announces the new integration between Amazon QuickSight Research and S&P Global, combining global energy news, research, and insights with comprehensive market intelligence to provide QuickSight Research customers with a deep research agent.

### [Blog 5 - How Contra Costa County District Attorney's Office modernized subpoena processing with AWS and CC Tech Digital](3.5-Blog5/)

This blog describes how the Contra Costa County District Attorney's Office partnered with AWS and CC Tech Digital to modernize their subpoena processing workflow using a serverless, cloud-native solution, managing over 17,000 subpoenas annually with improved speed, accuracy, and compliance.

### [Blog 6 - Streamlining collaboration across Departments of Motor Vehicles with Private Blockchain](3.6-Blog6/)

This blog discusses how blockchain can streamline driver's license issuance and foster deeper collaboration among DMVs across all 50 states, explaining why blockchain is an attractive technology choice for improving current reciprocity models.

### [Blog 7 - Introducing the AWS Infrastructure as Code MCP Server: AI-Powered CDK and CloudFormation Assistance](3.7-Blog7/)

This blog introduces the **AWS IaC MCP Server** (MCP standard), which enables AI assistants to help you search CDK/CloudFormation documentation, **validate templates**, run compliance checks, and **troubleshoot** deployment failures (with CloudTrail), with an emphasis on local execution for security.

### [Blog 8 - AWS Clean Rooms launches privacy-enhancing synthetic dataset generation for ML model training](3.8-Blog8/)

This blog announces the ability to generate **synthetic datasets** in **AWS Clean Rooms** for training ML models (regression/classification) while protecting privacy: preserving statistical patterns, reducing re-identification risk, providing privacy parameters, and reporting quality metrics (fidelity/privacy).

### [Blog 9 - AWS Partner Central now available in AWS Management Console](3.9-Blog9/)

This blog presents the **Partner Central experience integrated directly into the AWS Console**, creating a unified journey from AWS customer → Partner → Marketplace Seller, including key areas (Solutions, Profile, Opportunities, Partner connections), an identity foundation based on IAM/IAM Identity Center, and the role of APIs.

### [Blog 10 - Announcing AWS Partner Central in the AWS Management Console](3.10-Blog10/)

This blog announces **Partner Central in the AWS Console**, highlighting integration and automation via APIs (Account/Connections, Solution, Benefits, Selling/Leads), guidance using Amazon Q chat, access control through IAM/SSO, and expanded solution discovery in AWS Marketplace; it also includes steps for getting started and migration.

### [Blog 11 - Announcing Amazon EKS Capabilities for workload orchestration and cloud resource management](3.11-Blog11/)

This blog introduces **Amazon EKS Capabilities** (fully managed) to simplify workload orchestration and AWS/Kubernetes resource management, with launch capabilities such as **Argo CD**, **ACK**, and **KRO**, plus how to enable/configure them and key considerations around permissions, upgrades, and adoption.

### [Blog 12 - Simplify IAM policy creation with IAM Policy Autopilot, a new open source MCP server for builders](3.12-Blog12/)

This blog introduces **IAM Policy Autopilot** (an open source MCP server) that analyzes code to generate valid **IAM identity-based policies**, supports integration with AI coding assistants, updates permissions as code changes, and emphasizes reviewing/refining for **least privilege** before deployment.
